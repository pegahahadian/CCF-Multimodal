# CCF-Multimodal


## Multimodal Osteoarthritis Progression Prediction

This repository contains code, data files, and presentation materials for a multimodal model project developed at Cleveland Clinic Foundation (CCF). The project focuses on predicting osteoarthritis progression using clinical, demographic, radiographic, and MOAKS-related baseline features from the FNIH/OAI dataset.

The main goal of this project is to develop and evaluate multimodal models that can support risk stratification for osteoarthritis progression by integrating multiple data modalities.

---

## Project Overview


This project investigates whether multimodal baseline data can be used to predict OA progression categories. The repository includes:

- Baseline clinical and demographic variables
- Radiographic features
- MOAKS-related imaging-derived features
- Fusion modeling pipeline
- Data dictionary
- Presentation material summarizing the project
